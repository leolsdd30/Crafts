# Public: .htaccess

## Idea & Main Objective
The `.htaccess` file tells the Apache server (or XAMPP) how to handle incoming URLs.

## Purpose & Role
Cleans up URLs (changing `.com/search.php` to `.com/search`) and forces all traffic through `index.php`.

## Required Code & Logic
- `RewriteEngine On`
- `RewriteCond %{REQUEST_FILENAME} !-f` (If the requested file isn't an actual image or CSS asset...)
- `RewriteRule ^(.*)$ index.php [QSA,L]` (...force it to be handled by index.php).
