# View: Public Index (Home)

## Idea & Main Objective
The `index.php` view inside `/views/public` is the landing page of CraftConnect.

## Purpose & Role
Converts visitors into users. Provides a beautiful hero section, clearly states the value proposition, and pushes them toward the Search flow.

## Required Code & Logic
- Hero section with a massive unified search bar (using Tailwind CSS `bg-hero-pattern`).
- A "Browse by Category" grid (Plumbers, Electricians, Painters).
- A section outlining the "Trust & Verification" guarantee.
- Links to the `register.php` page targeting craftsmen ("Earn money on CraftConnect").
