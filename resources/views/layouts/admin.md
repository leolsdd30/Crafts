# Layout: Admin

## Idea & Main Objective
The `admin.php` layout is the master shell tailored specifically for internal tools.

## Purpose & Role
Provides a dense, sidebar-navigation focused UI strictly for the admin team.

## Required Code & Logic
- A fixed left sidebar layout (Tailwind `w-64 fixed h-screen`).
- No public footer.
- Links to Verifications, Disputes, Users, and Settings.
