# Config: app

## Idea & Main Objective
The `app.php` config holding general platform rules.

## Purpose & Role
Provides fallbacks and overarching settings like URL base strings, locales, and feature toggles.

## Required Code & Logic
- `name` => 'CraftConnect Platform'
- `env` => 'local' or 'production'
- `url` => 'http://localhost'
- `timezone` => 'UTC'
